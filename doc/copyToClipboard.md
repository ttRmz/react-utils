# copyToClipboard

## Usage

```jsx
import { copyToClipboard } from '@ttrmz/react-utils'

copyToClipboard('lorem ipsum', (error) => console.log('error : ', error))
```

## Parameters

| Name     | Description                              | Type     |
| -------- | ---------------------------------------- | -------- |
| text     |                                          | String   |
| callback | callback function that returns the error | Function |
